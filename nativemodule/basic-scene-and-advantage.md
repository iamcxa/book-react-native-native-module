# 適合情境與優勢

## 適合情境

![native-module-integration](assets/native-module-integration.png)

- 與硬體深度整合
- 需要使用 RN 沒有實作的 SDK
- 搭配已成熟的自家 SDK，使用 RN 快速迭代開發
- 介面與邏輯分層開發
- 整合已有原生專案＊

## 優勢

![native-module-integration-2](assets/native-module-integration-2.png)

- 套件行為完全可控
- 突破 RN 或第三方套件限制
- 保密性－程式可額外加密
- 提供軟硬整合新機會
- 縮短龐大專案迭代時程
